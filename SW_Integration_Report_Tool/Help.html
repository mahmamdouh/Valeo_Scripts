1 Introduction

1.1 Release note

This version is under user testing. It is an automatic report generator of integration reports that contain all the integrated SWC and changes to a specific project , for a specific release.

 

1.2 Purpose

Main purpose of this tool is to automate integration report generation.

the report contain all gerrit changes and its information like ( author , chane ID

commit message , date , Rolling builds , SWCs , functional affected SWCs,

list of changed files , polarion ticket with its status and changed files types ) .

 

Restrictions - important information

 

● Project po must be on your local machine

● local repo must refer to the latest head of your current release.
